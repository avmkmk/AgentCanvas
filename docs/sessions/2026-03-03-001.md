## GitLab MCP Gap Summary

**Available via MCP:**
- `search_repositories` — search existing repos
- `get_repository_tree` — browse file structure
- `read_file_contents` — read files
- `search_project_contents` — text search in project
- `create_project` — create new project

**Missing (use GitLab REST API via curl):**
- Issues: create, list, update, close, add labels
- Merge Requests: create, list, approve, merge
- Branches: create, list, delete, protect
- Labels: create, assign
- Milestones: create, assign
- CI/CD: trigger, view pipelines
- Members: add, remove, update permissions
- Webhooks: configure

**Workaround**: Use `curl` with `GITLAB_TOKEN` env var for all GitLab operations not covered by MCP.
